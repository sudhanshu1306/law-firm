# Law-Webapp
It is a job hunting and skills improving website built exclusively for law professionals. 

## Features
=> Job seekers can create an id and post their relevant skills to search for jobs.\
=> Articles and discussion forums to keep everyone updated about what's going on in the field of law.\
=> Showing relevant jobs to employees based on their preferences as well as showing relevant employees to employer based on the skills they are looking for in the job.\
=> Dashboard for both employees and employer to checkout and set their preferences, skills, education
details and records.\
=> Training and courses for employees to improve their skillsets.

## Tech Stack
MERN Stack

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/sudhanshu1306/law-firm.git`

### Setup

> now install npm packages

```shell
$ npm install
```
> run server
```shell
$ npm run dev
```

Server is running at `http://localhost:5000/`
